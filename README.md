# GEE4Ice
Google Earth Engine scripts for Snow/Ice dynamics over Greenland and Pyrenees

Some applications in snow/ice environments:
 - Anual NDSI evolution
 - Snow/Ice % masking 
 - Clip NDSI to only snow/ice cover
 - Export NDSIs to Drive
 - ...
